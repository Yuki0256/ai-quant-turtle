一、海龟交易策略概述

1.1 海龟策略的起源

海龟交易法则（Turtle Trading Rules）诞生于1983年，由著名商品交易员 Richard Dennis 和他的合伙人 William Eckhardt 共同创立。Dennis 坚信交易能力可以通过系统化训练来培养，为此他从1000多名申请者中挑选了13名"海龟"学员，教授他们一套完整的机械化交易系统。在随后的四年中，这批海龟学员创造了年均复合收益率超过80%的业绩，使海龟交易法则成为量化交易历史上最具传奇色彩的策略体系之一。

1.2 海龟策略的核心思想

海龟策略是一种纯粹的趋势跟踪（Trend Following）策略，其核心哲学包括四大支柱：

（1）顺势而为。海龟策略不预测市场方向，而是等趋势确立后跟随。当价格创出近期新高时，策略认为上升趋势已经形成，此时入场做多；当价格跌破近期低点时，策略认为上升趋势可能终结，此时平仓退出。

（2）让利润奔跑，截断亏损（Cut Losses Short, Let Profits Run）。海龟策略通过唐奇安通道突破入场捕捉趋势，同时使用基于 ATR 的严格止损纪律控制单笔交易的风险。单笔交易最大亏损被限制在账户总权益的2%以内。

（3）系统化执行。海龟策略是一套完全机械化的交易系统，所有买卖决策都基于明确的量化规则——何时入场、何时止损、何时退出，都有清晰的数学定义。这种系统化的方法消除了情绪对交易决策的干扰。

（4）分散化投资。原版海龟策略鼓励同时在多个不相关的市场中运行，通过多元化降低整体组合的波动性。

1.3 海龟策略的关键优势

① 规则清晰、完全量化：所有交易决策都有明确的数学公式，不存在主观判断空间，易于编程实现和自动化执行。

② 强大的风险控制：基于 ATR 的动态仓位管理和止损规则提供了天然的下行保护。即使连续亏损，账户最大回撤仍可控制在一定范围内。

③ 长期正期望值：虽然单笔交易胜率通常为35%-45%，但盈亏比可达2:1以上，使得长期期望值为正。盈利模式是"小亏多次 + 大赚少数"。

④ 适应性强：通过参数调整，海龟策略可以适应不同市场和不同时间周期。

⑤ 经过实战检验：海龟策略在40余年的历史中经历了多轮牛熊周期的考验，至今仍是众多专业量化基金的核心策略之一。

二、海龟策略核心概念解释

2.1 唐奇安通道（Donchian Channel）

唐奇安通道由 Richard Donchian 发明，是海龟策略中最重要的技术指标。它由两条线组成：上轨（Upper Band）= 过去 N 日的最高价；下轨（Lower Band）= 过去 N 日的最低价。

海龟策略使用两条不同周期的唐奇安通道：

系统一（入场通道）：周期为20日。当收盘价突破20日最高价（即上轨）时，产生买入信号，表示短期上升趋势确立，海龟入场做多。

系统二（退出通道）：周期为10日。当收盘价跌破10日最低价（即下轨）时，产生卖出信号，表示短期上升趋势可能终结，海龟平仓退出。

唐奇安通道的本质是"价格突破"思想：价格创短期新高意味着买方力量强劲，趋势可能延续；价格创短期新低意味着卖方开始主导，趋势可能反转。通过20日入场 + 10日退出的不对称设计，海龟策略实现了"快速入场、更快速退出"的非对称风险管理。

2.2 平均真实波幅（Average True Range, ATR）

ATR 由 J. Welles Wilder 在1978年提出，用于衡量市场波动性。与标准差不同，ATR 能更好地捕捉跳空缺口带来的额外风险。

计算步骤：

第一步——计算真实波幅（True Range, TR）：
TR = max(|H − L|, |H − Cprev|, |L − Cprev|)，其中 H 为当日最高价，L 为当日最低价，Cprev 为前一日收盘价。

第二步——使用 Wilder 平滑方法计算 ATR：
初始 ATR = 前 N 期 TR 的简单平均值；后续 ATR = (前一日ATR × (N−1) + 当日TR) / N。

在海龟策略中，ATR 扮演三个关键角色：① 仓位规模计算；② 止损设置（止损价 = 入场价 − 2 × ATR）；③ 加仓间距（每 0.5 ATR 为一个加仓单位）。

2.3 止损条件

止损是海龟策略风险管理的核心。海龟策略采用基于 ATR 的动态止损机制：止损价 = 入场价 − N × ATR（经典参数 N=2）。

这种止损机制的精妙之处在于：① 动态适应——高波动市场 ATR 大，止损带更宽，避免被正常波动"震出"；低波动市场 ATR 小，止损带更窄，快速截断亏损。② 风险量化——每笔交易的最大亏损被预先量化为账户权益的固定比例。③ 心理优势——明确的止损规则消除了"希望股价反弹"的侥幸心理，帮助交易员严格执行纪律。

三、Python 编程实现

3.1 数据加载

本策略使用 Tushare Pro API 获取的 A 股日线行情数据，以 CSV 格式存储。分析标的包括四只卫星通信及相关领域的 A 股：688418.SH 震有科技（通信设备）、600118.SH 中国卫星（卫星制造）、601698.SH 中国卫通（卫星通信服务）、300045.SZ 华力创通（卫星导航）。每只股票的数据包含 trade_date、open、high、low、close、vol、amount 等字段，加载后按日期升序排列。

3.2 唐奇安通道计算

使用 pandas 的 rolling 方法计算：通道上轨 = df["high"].rolling(20).max().shift(1)；通道下轨 = df["low"].rolling(20).min().shift(1)。注意使用 shift(1) 将通道值向后偏移一天，避免使用当日高/低价判断当日收盘价，这是避免"未来函数"偏误的关键步骤。

3.3 ATR 计算

按照 Wilder 的方法，首先计算 True Range（真实波幅），然后使用递归平滑公式计算 ATR。ATR 的值同样使用 shift(1) 偏移一天，确保回测中使用的 ATR 只基于历史数据。

3.4 交易信号生成

信号逻辑简洁明确：买入信号——当日收盘价 > 20日唐奇安通道上轨；卖出信号——当日收盘价 < 10日唐奇安通道下轨。回测中，买入信号出现时以收盘价建仓；卖出信号出现时以收盘价清仓。止损条件独立于信号系统，当价格跌破"入场价 − 2 × ATR"时强制平仓。

3.5 回测与绩效指标

回测引擎模拟完整交易流程，初始资金设为100,000元。每次买入时根据 ATR 和风险预算计算仓位。回测结束后计算以下核心绩效指标：

累计回报率 = (最终资产 / 初始资产 − 1) × 100%

年化收益率 = 累计回报率(252/交易天数) − 1

最大回撤 (MDD) = min((当日资产 / 历史峰值 − 1)) × 100%

夏普比率 = (日收益率均值 / 日收益率标准差) × √252

年化波动率 = 日收益率标准差 × √252

胜率 = 盈利交易次数 / 总交易次数 × 100%

七、海龟法则的适应场景与使用心得

7.1 适应场景

① 强趋势市场：在明显的单边牛市或熊市中，海龟策略能够最大化趋势收益。趋势越长、越持续，策略的累计收益越高。② 中长线投资周期：海龟策略是典型的中线趋势跟踪策略（持仓周期通常为几周至几个月），不适合日内交易。③ 高流动性标的：交易活跃、买卖价差小的标的能降低滑点和冲击成本。④ 多品种组合：当在多个低相关性的品种上同时运行时，组合层面的风险调整收益显著优于单一品种。

7.2 不适用场景

① 剧烈震荡市场：价格在窄幅区间反复波动时，通道突破产生大量假信号，频繁止损导致策略净值持续回撤。② 突发反转行情：当价格因突发事件快速反转时，策略来不及反应。③ 流动性枯竭的市场：成交量极低的标的的滑点问题会严重侵蚀策略收益。

7.3 使用心得与感悟

第一，简单策略蕴含深刻智慧。海龟策略的逻辑只有寥寥几条规则——突破20日高点买入，跌破10日低点卖出，止损2倍ATR——但背后凝聚了趋势跟踪交易哲学的精髓。量化交易的核心不在于模型的复杂程度，而在于逻辑的正确性和执行的一致性。

第二，风险控制是交易的生命线。海龟策略最令人震撼的不是它的盈利公式，而是它的风控体系。基于ATR的动态仓位管理和2%止损规则，确保了策略在任何市场环境下都能"活下去"。在这个市场上，首先要考虑的不是赚多少，而是能活多久。

第三，接受亏损是盈利的前提。海龟策略的胜率通常在35%-45%之间——超过一半的交易都是亏损的！但正是这些"小亏损"为少数"大盈利"创造了条件。这颠覆了"高胜率=好策略"的直觉认知，让人理解盈亏比的重要性。

第四，纪律是策略有效的前提。回测中严格按信号执行与实盘中看到浮亏就恐慌平仓，两者的结果天壤之别。知道和做到之间隔着人性的鸿沟。

第五，没有圣杯，只有适应。海龟策略的表现高度依赖市场状态。在趋势市场中它是利器，在震荡市场中它是钝刀。量化交易的本质是以数据为基础做判断，而非盲目相信某一个"神奇公式"。
