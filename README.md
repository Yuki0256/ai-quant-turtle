# 🐢 海龟交易法则 — 量化交易 TASK4

> 复刻传奇：基于唐奇安通道 + ATR + 风险预算的趋势跟踪策略实战

## 项目概述

本项目是量化交易工作坊的第四个任务（TASK4），完整实现了经典的海龟交易法则（Turtle Trading Rules），包括：

- **唐奇安通道 (Donchian Channel)**：20日入场通道 + 10日退出通道
- **平均真实波幅 (ATR)**：Wilder 递归平滑方法，用于仓位管理和止损
- **交易信号生成**：突破入场 / 跌破退出 / 止损平仓
- **回测引擎**：模拟交易 + 绩效评估（MDD、夏普比率、累计回报等）
- **可视化分析**：多面板图表，股价、通道、信号、资金曲线、回撤、ATR
- **参数调优**：不同通道周期、不同股票的对比分析

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行策略回测

```bash
python turtle_strategy.py
```

这将依次执行：
1. 加载四只股票的日线数据
2. 以震有科技为例，运行经典海龟策略（20/10日通道）回测
3. 测试 6 组不同的通道周期参数组合进行对比
4. 对比 4 只股票在相同参数下的表现
5. 生成所有可视化图表到 `images/` 目录
6. 输出回测汇总 CSV 到 `reports/` 目录

### 3. 生成 PDF 报告

```bash
python generate_report.py
```

生成符合提交要求的 PDF 报告（宋体/黑体、五号字、1.5倍行距、两端对齐）。

## 策略说明

### 海龟交易法则核心理念

海龟交易法则由 Richard Dennis 和 William Eckhardt 在 1983 年创立，是量化交易历史上最著名的趋势跟踪策略。其核心哲学是：

- **顺势而为**：不预测市场方向，等趋势确立后跟随
- **让利润奔跑，截断亏损**：通过通道突破捕捉大趋势，通过严格止损控制风险
- **系统化执行**：完全机械化的交易规则，消除情绪干扰

### 核心参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 入场通道周期 | 20 日 | 唐奇安通道上轨突破 = 买入 |
| 退出通道周期 | 10 日 | 唐奇安通道下轨跌破 = 卖出 |
| ATR 周期 | 20 日 | Wilder 平滑方法 |
| 止损倍数 | 2 × ATR | 止损价 = 入场价 - 2 × ATR |
| 风险预算 | 1% | 单笔最大亏损占账户比例 |

### 绩效指标

| 指标 | 说明 |
|------|------|
| 累计回报率 | 最终资产 / 初始资产 - 1 |
| 年化收益率 | 累计回报率年化 (252交易日) |
| 最大回撤 (MDD) | 资金曲线最大百分比回撤 |
| 夏普比率 | 风险调整后收益 (无风险利率=0) |
| 年化波动率 | 收益率的年化标准差 |
| 胜率 | 盈利交易次数占总交易次数的比例 |

## 自定义分析

在 `turtle_strategy.py` 中修改参数即可进行自定义分析：

```python
# 修改通道周期
result = backtest_turtle(df, entry_period=30, exit_period=15)

# 修改止损倍数
result = backtest_turtle(df, stop_multiple=3.0)

# 修改风险预算
result = backtest_turtle(df, risk_fraction=0.02)
```
